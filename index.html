Simplified syntax of math functions and defines

You can now use short names for mathematical functions:

  • sin(x) instead of Math.sin(x)
  • cos(x) instead of Math.cos(x)
  • tan(x) instead of Math.tan(x)
  • abs(x) instead of Math.abs(x)
  • sqrt(x) instead of Math.sqrt(x)
  • and many others...

Types of output

Bytebeat (0 to 255): Classic bytebeat, output value from 0 to 255.

Floatbeat (-1 to 1): output value from -1.0 to 1.0, suitable for good quality.

Signed Bytebeat (-128 to 127): Classic bytebeat, but with signed output, the output value is from -127 to 128.


About Bytebeat

Bytebeat is a technique for creating music using very short programs, usually a single line of code, where t is time.

Bytebeat example

t * ((t >> 12 | t >> 8) & 63 & t >> 4)

Floatbeat example

sin(t * 0.001) * 0.5 + 0.5

Signed Bytebeat example

(t >> 5 | t >> 8) * ((t >> 4) & 63) - 64

Difficult melody

(t >> 7 | t | t >> 6) * 10 + 4 * (t & t >> 13 | t >> 6)

Sine example

(sin(t * 0.001) * 64 + 128) & (t >> 4)

Exponential example

(t * log(t * 0.001) & 255) * sin(t * 0.0001)
